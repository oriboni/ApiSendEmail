-Название проекта

Веб-сервис для отправки почты.

-Описание проекта

Сервис предназначен для отправки почты.

-Целевая аудитория

Сервис предназначен для разработчиков. Он позволяет не писать свой код для отправки почты, а пользоваться данным сервисом.

-Основные функции API:

Отправка письма: Отправка обычного письма с темой и текстом.

Отправка письма со скрытой копией: Отправка простого письма с темой и текстом, а так же отправка скрытой копии.

Отправка письма с копией: Отправка простого письма с темой и текстом, а так же отправка копии.

Отправка письма с html кодом: Отправка письма с темой и текстом, а так же возможностью написания кода.

-Архитектура и используемые технологии

Язык программирования: node.js

Фреймворки: express

Библиотеки: nodemailer для отправки почты, swagger для документации
